# Egymove

Egymove is an Egypt-focused logistics marketplace website for connecting customers with drivers and vehicles for parcels, retail deliveries, furniture, and scheduled moves.

## Run locally

```powershell
npm start
```

Or run the server directly:

```powershell
node server.js
```

Then open `http://localhost:4173`.

Do not open `index.html` directly with `file://`. The app uses JavaScript modules, so it should be served through the local server URL.
